## javascript let ##
```javascript
// variable declared using let
let name = 'Sara';
{
    // can be accessed only inside
    let name = 'Peter';

    console.log(name); // Peter
}
console.log(name);
```
## javascript const ##
```javascript
// name declared with const cannot be changed
const name = 'Sara';
```
